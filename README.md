# Config

Configuration files for [@yous](https://github.com/yous)

## Requirements

* [Vundle][]

[Vundle]: https://github.com/gmarik/vundle

## Install

``` sh
$ ./install.sh
$ git submodule init
$ git submodule update
$ git config --global core.excludesfile ~/.gitignore
$ gem install what_methods wirble
$ chsh -s /bin/zsh
```

In Vim,

```
:BundleInstall
```
