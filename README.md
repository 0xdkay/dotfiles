# Requirements

* [Vundle][]

[Vundle]: https://github.com/gmarik/vundle

# Install

    $ ./install.sh
    $ git submodule init
    $ git submodule update
    $ git config --global core.excludesfile ~/.gitignore
    $ gem install what_methods wirble
    $ chsh -s /bin/zsh

In Vim,

    :BundleInstall
